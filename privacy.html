🔒 Коротко: РуссКопилот не собирает, не хранит и не передаёт ваши личные данные. Всё остаётся в вашем браузере.

1. Общие положения

Настоящая политика конфиденциальности описывает, как расширение РуссКопилот («Расширение», «мы») обрабатывает информацию при использовании Chrome-расширения.

2. Какие данные мы НЕ собираем

Расширение РуссКопилот не собирает:

  • Личную информацию (имя, email, телефон, адрес)
  • Историю браузера или посещённые страницы
  • Тексты страниц, которые вы анализируете
  • Ваши вопросы и ответы AI
  • API-ключи (хранятся только локально)
  • Данные банковских карт или платёжную информацию

3. Что хранится локально

В хранилище вашего браузера (chrome.storage.local) сохраняются только:

  • Настройки расширения (включение/выключение функций)
  • Счётчик дневных запросов (сбрасывается каждые 24 часа)
  • API-ключ OpenAI (если вы его указали) — только локально
  • Статус подписки

Эти данные не передаются на наши серверы и доступны только вам.

4. Передача данных третьим лицам

Когда вы используете AI-функции расширения, текст страницы передаётся напрямую в API OpenAI (api.openai.com) по вашему собственному ключу. РуссКопилот не является посредником — данные идут напрямую от вашего браузера к OpenAI согласно их политике конфиденциальности.

5. Разрешения расширения

Расширение запрашивает следующие разрешения Chrome:

  • activeTab — для чтения текста текущей вкладки только по вашему запросу
  • storage — для хранения настроек локально в браузере
  • scripting — для отображения кнопки на страницах

6. Дети

Расширение не предназначено для лиц младше 13 лет. Мы сознательно не собираем данные детей.

7. Изменения политики

Мы можем обновлять эту политику. Актуальная версия всегда доступна на этой странице. При существенных изменениях мы уведомим пользователей через расширение.
