🔒 Коротко: АнтиФишинг Рунет не собирает, не хранит и не передаёт ваши личные данные на наши серверы. Всё остаётся в вашем браузере.

1. Общие положения

Настоящая политика описывает, как Chrome-расширение АнтиФишинг Рунет обрабатывает информацию при использовании.

2. Данные, которые мы НЕ собираем

Расширение не собирает: личную информацию, историю браузера, платёжные данные, пароли или учётные данные.

3. Локальное хранение

В хранилище браузера (chrome.storage.local) сохраняются только настройки расширения и статус подписки. Эти данные не передаются на наши серверы.

4. Передача данных

Расширение проверяет только доменное имя открываемого URL по локальной базе паттернов. URL не сохраняются, не логируются и не передаются. Сайты, о которых сообщает пользователь, сохраняются только локально.

5. Разрешения расширения

  • activeTab — Для получения URL текущей вкладки с целью проверки по базе угроз
  • storage — Для хранения счётчика угроз, белого списка и настроек
  • scripting — Для встраивания красного баннера на опасные страницы
  • webNavigation — Для отслеживания загрузки новых страниц
  • notifications — Для уведомлений при обнаружении угрозы (Pro)

6. Дети

Расширение не предназначено для лиц младше 13 лет.
